Jackal - как пример приложения, которое с самого начала разрабатывается как TDD. Пример старый, поэтому некоторых библиотек нет в репозитории. Так, присутствуют самописные расширения для DSL для тестов, чтобы был чистый английский язык в тестах. 
Nightmare - просто попугать народ. Пусть поковыряются и ужаснутся. Пусть подумают, как такое вообще можно тестировать, с чего начинать писать тесты. 
NUnitIntroduce - краткие примеры на возможности NUnit вообще. Чтобы не забыть про всякие источники данных. Просто полезно будет раздать или напомнить самому себе. 

Остальной код слушателями пишется с нуля, заготовок никаких нет.
